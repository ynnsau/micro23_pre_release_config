# micro23\_pre\_release\_config
This repository contains the configurations and steps to reproduce figures in paper `Demystifying CXL Memory with True CXL-Ready Systems and CXL Memory Devices (MICRO'23)`. 

## Before you start
Please clone the repository to test `memo` and `Caption`:
```

```
